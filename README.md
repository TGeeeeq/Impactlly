# Impactlly - Impact Tracking Platform

## Overview

Impactlly is a platform designed to help individuals and organizations track, measure, and visualize their environmental and social impact. The platform provides tools for logging activities, measuring outcomes, and sharing impact stories.

## Features

- Impact tracking and measurement
- Data visualization and reporting
- Goal setting and progress tracking
- Community sharing and collaboration
- Mobile-responsive design

## Technology Stack

- **Framework**: Next.js 16.1.6
- **Styling**: Tailwind CSS 4.2.0
- **UI Components**: Custom component library with Radix UI primitives
- **State Management**: React Hook Form
- **Database**: Supabase
- **AI Integration**: Vercel AI SDK
- **Maps**: Leaflet with React Leaflet
- **Charts**: Recharts
- **Animations**: Framer Motion
- **Form Validation**: Zod
- **TypeScript**: 5.7.3

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

4. Start the production server:
   ```bash
   npm run start
   ```

## Design System

Impactlly features a nature-inspired design system with:

- **Color Palette**: Earth-inspired colors including Earth Green, Soil Brown, Sky Blue, Sun Yellow, Leaf Green, and Clay Red
- **Typography**: Fluid typography with serif headings and sans-serif body text
- **Component Variants**: Extended variants for buttons, inputs, textareas, and cards including organic, outline, raised, and nature styles
- **Responsive Design**: Mobile-first approach with adaptive layouts

## Component Library

The project includes a comprehensive UI component library located in `components/ui/` with:

- Buttons with multiple variants (default, destructive, outline, secondary, ghost, link, organic, pulse, float)
- Inputs with variants (default, outline, flushed, underlined, organic)
- Textareas with variants (default, outline, flushed, underlined, organic)
- Cards with variants (default, outline, raised, organic, nature) and size options
- And many other components following consistent design patterns
